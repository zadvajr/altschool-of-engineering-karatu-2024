HTML DOM Document

The HTML DOM Document object represents the whole HTML document.

It is the owner of all other objects in your web page.

The HTML DOM Document object has various properties and methods that allow you to create, modify, and delete elements and attributes in your web page.

Finding HTML Elements

  • document.getElementById('demo') - finds an element by element's ID
  • document.getElementByTagName('p') - finds an element by their tag names.
  • document.getElementByClassName() - finds and element by their class name.

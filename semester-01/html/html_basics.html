HTML Basics

HTML Headings

HTML headings are defined with <h1> to <h6> tags

<h1> defines the most important heading. <h6> defines the least important heading.

HTML Paragraphs

HTML Paragraphs are defined with <p> tag.

For example the following are paragraphs

<p>This is a paragraph</p>

<p>This is another paragraph</p>

HTML Links

HTML Links are defined with the anchor <a> tag:

For example: <a href="https://www.w3schools.com">W3schools<a>

The links destination is specified in the href attribute

Attributes are used to provide additonal information about HTML elements.

HTML Lists

There are broadly two types of HTML Lists, unordered list and ordered lists.

Unordered List

Example as follows

  • Apple
  • Banana
  • Cherry
  • Dragon Fruit

Ordered List

This is a numbered list format

  1. Apple
  2. Banana
  3. Cherry
  4. Dragon Fruit

Description List

Description list is a list of terms with each definition

Apple
- Apple is a fruit
Banana
- Banana is a fruit
- Cherry
- Cherry is a fruit
Dragon Fruit
- Dragon Fruit is a fruit
